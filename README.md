IronTester
==========

Exemplary use of NServiceBus for PG Technical University Classes - Code builds pipeline and tests runner

## Used technologies
* NServiceBus - 4.6.1
* MSMQ
* RavenDb - 2.330
* .NET 4.5.1 (C# & MVC5)
* SbAdmin - 2.0
* SingalR - 2.1.2
* AngularJs - 1.2.25

## Testing enviorment
* Windows 8.1 x64 PRO ENG
